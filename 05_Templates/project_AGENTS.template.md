# Project AGENTS Template

## Project Memory

Before starting non-trivial work:

1. Read this project's `README.md`.
2. Read local `docs/ai-memory/` if it exists.
3. Read the shared YUSU knowledge vault:
   - Windows: `F:\AcademicHub\0#YUSU\00_START_HERE_FOR_CODEX.md`
   - Ubuntu/WSL: `$YUSU_KB_ROOT/00_START_HERE_FOR_CODEX.md`
4. Search the shared vault for this project name, repository path, framework, hardware, and recurring errors.

After non-trivial work:

1. Update local project memory if present.
2. Add reusable experience to the shared vault under `01_Projects/<project-slug>/`.
3. Promote cross-project lessons to `03_CrossProject/` only when they are stable or clearly reusable.
4. Never store secrets in memory files.
