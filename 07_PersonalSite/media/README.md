# Personal Site Media Library

This directory is the formal media store for the local YUSU personal site.

## Structure

- `raw/awards/`: original award certificates and certificate scans.
- `raw/documents/`: original public award-list PDFs/DOCX files.
- `raw/reference/`: original design/reference video files.
- `derived/reference/`: lightweight derived reference artifacts such as contact sheets.

## Rules

- Preserve raw files without destructive edits.
- Put display-only derivatives under `derived/` or `web/assets/`.
- Update `07_PersonalSite/notes/materials-inventory.md` and `07_PersonalSite/data/showcase.json` when adding new media.
- Review privacy before exposing this local site outside `127.0.0.1`.
