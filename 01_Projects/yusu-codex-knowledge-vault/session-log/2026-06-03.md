## Reusable Lessons

- Do not call a Windows folder “shared with Ubuntu” until Ubuntu has actually mounted/cloned and written back.
- Prefer GitHub private sync for cross-boot availability when direct filesystem sharing is not verified.
- Newly installed skills may need a fresh Codex discovery surface; validate files and links, but do not overclaim passive trigger in the current long thread.
- A knowledge system should write its own project memory early; otherwise it stays a nice shell with no lived evidence.
