## 定位

Auto Play 是一个本地 Windows 游戏自动化工具。它从通用图像识别点击器，演进为更实用的鼠标键盘宏录制器和高频键盘连点器。当前默认面向用户的工具是由 `app.py` / `start.bat` 启动的 Tkinter 宏录制器。

最重要的产品经验：当游戏流程固定且可重复时，可靠的宏录制器通常比复杂的图像识别/状态机更适合作为第一版可用交付。
