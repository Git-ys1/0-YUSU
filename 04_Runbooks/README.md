# Runbooks

本目录保存共享知识库的操作协议。后续 Codex 做知识入库、检索、跨系统同步或成熟项目复盘时，应先读这里的相关文件。

## Core Runbooks

- `system-decisions.md`: 共享目录、同步方式、记忆写入层级的系统级决策。
- `codex-retrieval-workflow.md`: Codex 如何搜索和使用本知识库。
- `codex-ingestion-guide.md`: 普通项目和通用经验的入库流程。
- `mature-project-ingestion.md`: 成熟项目的多轮证据复盘流程。
- `super-yusu-v0.2-ingestion.md`: 成熟项目最终总结与 `Memory Routing Audit` 协议。
- `super-yusu-v0.3-marginalia.md`: Marginalia 深度研究层、DeepSeek/BGE 路由、个人站源码级单进程集成和使用边界。

## Supporting Runbooks

- `windows-ubuntu-shared-folder.md`: Windows/Ubuntu 双系统共享与同步。
- `github-sync.md`: GitHub 私有仓库同步。
- `global-agents-snippet.md`: 全局 `AGENTS.md` 片段。
- `obsidian-setup.md`: Obsidian vault 使用方式。
- `completion-audit.md`: 入库完成后的通用检查。
